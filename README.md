# TestProvincia
Esta es una aplicación web API desarrollada en .NET 8 que proporciona solucion para el challenge enviado por provinciaNet. En la solución se usan EF Code First, por lo tanto hay migraciones para crear la Base de datos e insertar datos en la misma.
